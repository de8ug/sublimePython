# PyCharm

# 快速开始

01 打开PyCharm，单击“file”，然后选中“New Project..." .

02 在弹出的”New Project" 窗口中选则文件夹，如图红色方框位置。

03 选择后的如图所示，这里的文件夹随意，但最好命名全为英文，且在非C盘。

04 然后单击“Creat"。创建工程。

05 弹出的窗口可以选择默认，然后进入下一步。

06 创建完成。

07 现在，将鼠标放在创建的工程上，单击鼠标右键，然后依次选择“New>Python File".

08 然后会弹出一个”New Python File"的窗口，输入名字后单击“OK”。

09 然后，就可以输入代码了。

10 编写好以后，在编写代码的窗口中单击右键，然后选择“Run （你的文件名）”，就可以运行程序了。

11 在界面的下端就可以看到运行的结果。

# 常用设置：

  1. pycharm默认是自动保存的，习惯自己按ctrl + s 的可以进行如下设置：
（1）. file -> Setting -> General -> Synchronization -> Save files on frame deactivation 和 Save files automatically if application is idle for .. sec 的勾去掉
（2）. file ->Setting -> Editor -> Editor Tabs -> Mark modified tabs with asterisk 打上勾
1
2

  1. Alt + Enter: 自动添加包

  2. 对于常用的快捷键，可以设置为visual studio(eclipse...)一样的： file -> Setting -> Keymap -> Keymaps -> vuisual studio -> Apply

  3. Pycharm中默认是不能用Ctrl+滚轮改变字体大小的，可以在file -> Setting ->Editor-〉Mouse中设置

  4. 要设置Pycharm的字体，要先在file -> Setting ->Editor-〉Editor中选择一种风格并保存，然后才可以改变

  5. 在setting中搜索theme可以改变主题，所有配色统一改变
